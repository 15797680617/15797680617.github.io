docker for mac问题记录

1：数据卷路径无法找到

在linux上，使用docker创建数据卷：docker volume create –name v1
查看数据卷信息：docker inspect v1

其中”Mountpoint”: “/var/lib/docker/volumes/v1/_data” 指的就是数据卷的实际目录。
在linux上可以直接进入到数据卷的目录，但是mac上…

经过百度了解到，docker在mac上运行实际也是跑了一个linux的虚拟机，数据卷文件不会直接放在宿主机上，经过查阅相关资料，发现可以打开docker虚拟机窗口并找到它。
